# ADR 001: Switching from MkDocs to Zensical

**Date:** 2026-02-21
**Status:** Accepted

## Context
MkDocs 2.0 was announced with breaking changes that are incompatible with the Material theme. MkDocs 1.x is no longer maintained.

## Decision
We will use **Zensical** as the static site generator for ThePlatforge.

## Consequences
* **Positive:** Better performance, continued support for Material design, and a more stable live-reload experience.
